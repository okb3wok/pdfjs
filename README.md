# Личный кабинет с простой электронной подписью

## О проекте

Это веб-приложение — личный кабинет, где пользователи могут:

- Регистрироваться и авторизовываться.
- Подписывать документы простой электронной подписью.
- Управлять своими документами и настройками.

---

## Структура проекта

````
src/
├── css/        # Стили
├── js/         # Клиентские скрипты
├── php/        # Серверная часть на PHP
│ ├── app/      # Логика приложения (контроллеры, модели)
│ ├── public/   # Точка входа (index.php и др.)
│ └── db.json   # Файл с данными пользователей и документов (простая БД)
static/         # Статические ресурсы (изображения, шрифты)
templates/      # Шаблоны страниц
````
## Установка и запуск

1. Установить зависимости PHP:
   ```bash
   composer install
   ```
   
2. Установить зависимости JavaScript:
   ```bash
   npm install
   ```


## Технологии

- PHP 8+ (с использованием Slim Framework)
- JavaScript (для фронтенда)
- TailWindCSS
- JSON-файл как простая база данных
- Composer для управления зависимостями

## Возможные улучшения

- Использовать реляционную базу данных (MySQL, PostgreSQL) вместо JSON.
- Добавить полноценную аутентификацию с JWT или сессиями.
- Расширить поддержку электронных подписей.
- Добавить валидацию и проверку документов.
- Интеграция с внешними сервисами хранения и подписания.
